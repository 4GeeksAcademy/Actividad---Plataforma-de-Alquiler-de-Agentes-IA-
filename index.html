¿Qué es una Skill?

Las Skills son capacidades especializadas que pueden ser instaladas en los agentes de IA. Cada skill proporciona funcionalidades específicas como integración con bases de datos, acceso web, análisis de documentos, etc. Los agentes pueden tener múltiples skills habilitadas simultáneamente, y cada skill puede estar disponible para varios agentes en la plataforma.
